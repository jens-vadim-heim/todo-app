# ToDo-app

This is a simple todo-app developed using HTML, CSS and Javascript (React), built using Vite. 

## How-to

To write a todo, type some text in the input field. When ready, press enter/the "ADD" button to create the todo and render it. 

To remove a todo, press anywhere on the todo to remove it.
