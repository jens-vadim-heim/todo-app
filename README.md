# ToDo list

This is a simple "todo list" web application developed using HTML, CSS and Javascript (React), built using Vite. 

It allows the user to type in a note ("todo"), then add it to the list of todos. 

The user can manually mark todos as completed, and delete individual todos/each completed todo.

## How-to

### Create todo
Type some text in the input field. When finished, press enter/the "ADD" button to create the todo and display it. 

### Mark as completed
To mark a todo as completed, click inside of the todo to change its status and appearance.

### Delete individual todo
To manually delete a single todo, hover over it and press the "Delete" button in the top right corner of the todo. This can be done regardless of whether it is marked as completed or not.

### Delete *all* completed todos
To delete all todos marked as completed, press the red "DEL" button next to the "ADD" button.
